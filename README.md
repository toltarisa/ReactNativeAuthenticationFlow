### Projeyi Kurmak için

- repo'yu klonla ,cd ile kök dizine gir.
- yarn veya npm install ile paketleri kur
- .env dosyası oluştur ve içerisine FIREBASE_API_KEY 'i gir.
- Uygulamayı react-native run-android ile çalıştır.
